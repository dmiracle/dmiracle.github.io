# Portfolio Site

## Plan

Make a portfolio site using tailwind css and some other stuff. The basic idea is a header and a badge for each portfolio item. 

- Create simple html sample
- Add the bootstrap cdn
- Nav
- ~~Badges~~ Cards
- About me

## Steps

1. Download bootstrap template from [github bootstrap repo](https://github.com/twbs/bootstrap) in the site/docs/4.1/examples folder [^](https://stackoverflow.com/questions/17180364/how-can-i-download-the-examples-from-twitter-bootstrap-website)

    ```bash
    git clone https://github.com/twbs/bootstrap
    cp -r bootstrap/site/docs/4.1/examples/album dmiracle.github.io/
    ```
2. Add CDN to index

3. Added holder.js
4. Used Album template
5. Modify Cards for each project
6. Add icons for project links
